Contrôles

Cliquer sur une couleur pour la sélectionner, puis cliquer sur la grille pour dessiner.

Cliquer-droit sur la grille pour déplacer la grille.

Utiliser la molette de la souris pour zoomer.

Appuyer sur espace pour changer la couleur sélectionnée. Après avoir sélectionné une couleur, appuyer à nouveau sur espace pour fermer le menu.
